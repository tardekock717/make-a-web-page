LESSON 1

The Web and HTML (Hyper Text Markup Language):

What is the web is and how does it work?

The web is a collection of HTML documents that form the basis of almost every web page, HTML glues everything together. The major pieces of the web are

  1. YOU (your computer with a running browser)
  2. Internet
  3. Servers

Your computer sends http requests to the servers, the server then finds the appropriate HTML code and sends back to the browser (computer), your browser then interprets the HTML and displays this on the screen

HTML is the main type of document used on the web, you also get different types of files on the web such as plain text, images, videos and music.

HTML is made is made up of CONTENT(what you see);MARKUP(what the content looks like or how its arranged);LINKS(connect to other pages) and VIDEOS/IMAGES

LESSON 2

Creating a structured document with HTML:

How does CSS work with HTML?

HTML controls the structure of the page and CSS controls the style of the page (how it looks). HTML is a language and has syntax and rules. Basic word in HTML is tag. Browser turns the tags into elements that form tree like structure called the DOM

To define style you need to use the language CSS. CSS allows you to use specific syntax and rules to change how elements look on a page, eg: size, font, etc.

To create structured content is put content into boxes using "div" tags.

What is the DOM?

Dom stands for document object model. DOM is a standard convention for representing and interacting with elements in HTML. Basically a DOM tree is what the browser built from an HTML document.
